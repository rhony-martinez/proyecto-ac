◆ moverServo()

void moverServo ( )

Tarea asíncrona para control del servomotor.

Controla el movimiento continuo del servomotor.

Genera movimiento suave y continuo del servomotor

Genera movimiento oscilatorio suave del servomotor entre 20° y 90°:

  • Movimiento incremental de 20° a 90° (apertura)
  • Movimiento decremental de 90° a 20° (cierre)
  • Control asíncrono con intervalo de 15ms para movimiento fluido
  • Utilizado para simular control de persianas o ventilación

@usage Estado PMV_BAJO para cerrar persianas y conservar calor

void toggleBuzzer ( )

Tarea asíncrona para control del buzzer.

Genera patrón de sonido intermitente para el buzzer.

Genera patrón de sonido intermitente en estado ALARMA

Alterna el estado del buzzer cada 200ms creando un patrón de sonido de alerta. Utilizado en estado ALARMA para notificación audible de condiciones críticas.

@usage Estado ALARMA para alertas sonoras
